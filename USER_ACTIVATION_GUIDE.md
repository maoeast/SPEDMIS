# 用户激活指南

## ✅ 用户需要做的：非常简单！

### 激活流程只需3步

#### 步骤1：安装软件
1. 下载应用程序安装包：`特殊教育多模态干预系统 1.0.0.exe`
2. 双击运行安装程序
3. 按照提示完成安装
4. **不需要配置任何环境变量或密钥文件**

#### 步骤2：获取机器码
1. 首次启动应用时，会显示"激活"界面
2. 点击"**获取机器码**"按钮
3. 机器码会显示在界面上（例如：`a1b2c3d4e5f6...`）
4. 复制这个机器码

#### 步骤3：获取激活码并激活
1. 将机器码发送给软件开发者
2. 开发者使用激活码生成工具生成对应的激活码
3. 开发者将激活码返回给你（例如：`64位十六进制字符串`）
4. 在激活界面输入开发者提供的激活码
5. 点击"**激活**"按钮
6. 如显示"激活成功"，则应用已激活完成！
7. 应用会自动加载主界面，可以正常使用所有功能

**完成！你不需要做任何其他操作。**

## 🔒 激活安全说明

### 激活码的工作原理
- **激活码是唯一的**：每台电脑的机器码不同，对应的激活码也不同
- **激活码是不可伪造的**：基于加密算法生成，用户无法自己生成
- **激活信息已加密**：激活成功后，你的电脑激活信息以加密形式存储，无法被篡改

### 你的隐私
- 机器码仅用于生成激活码，不包含你的个人信息
- 机器码和激活码之间的对应关系由开发者安全保管
- 你的电脑上存储的激活信息已加密，即使他人获得也无法识别

## 🆘 常见问题

### Q: 如果我重装系统或更换电脑怎么办？
**A:** 重装系统或更换电脑后，机器码会改变。你需要：
1. 重新获取新的机器码
2. 联系开发者，提供新的机器码
3. 获得新的激活码，重新激活即可

### Q: 激活码可以在多台电脑上使用吗？
**A:** 不可以。每个激活码只对应一台电脑（由其机器码唯一确定）。如果需要在多台电脑上使用，需要为每台电脑分别生成激活码。

### Q: 如果激活失败，提示"激活码无效"怎么办？
**A:** 请检查：
1. 激活码是否复制完整且正确（无空格、无多余字符）
2. 激活码是否与你的机器码对应（需要确认提供的机器码和激活码是匹配的）
3. 系统时间是否正确（激活系统依赖时间戳）
4. 联系开发者，确认激活码是否生成正确

### Q: 我可以看到应用的密钥或激活信息吗？
**A:** 不可以，也不需要。应用会自动管理所有密钥和激活信息，用户无需接触。

### Q: 为什么应用要求激活？
**A:** 激活系统的目的是：
1. **防止盗版**：只有获得授权的用户才能使用应用
2. **保护用户**：激活过程验证应用的完整性，防止被篡改
3. **管理用户**：开发者可以跟踪应用使用情况，提供更好的支持和更新

## 📞 获取帮助

如果激活过程中遇到问题，请：
1. **记录机器码**：在获取机器码界面拍照或复制保存
2. **联系开发者**：提供机器码，说明遇到的问题
3. **等待回复**：开发者会生成激活码并返回给你

**记住：用户不需要任何技术配置，只需按照上述简单步骤操作即可。**
