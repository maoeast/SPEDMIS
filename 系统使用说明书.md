## 快速开始

### 环境准备

**最低配置要求：**

| 项目 | 要求 | 说明 |
|-----|------|------|
| 操作系统 | Windows 10/11 | 推荐使用64位系统 |
| 处理器 | 双核 2.0GHz 及以上 | 建议四核 |
| 内存 | 4GB 及以上 | 建议8GB |
| 硬盘空间 | 500MB 可用空间 | 用于安装应用及数据存储 |
| 分辨率 | 1024×768 及以上 | 推荐使用1920×1080 |

**一键检测脚本（可选）：**

```powershell
# 在PowerShell中运行，检查系统配置
[System.Environment]::OSVersion.Version
Get-WmiObject -Class Win32_ComputerSystem | Select-Object NumberOfProcessors
Get-WmiObject -Class Win32_PhysicalMemory | Measure-Object -Property Capacity -Sum
```

### 5分钟部署指南

#### 步骤1：获取安装包

联系供应商获取《特殊教育多模态干预系统》安装程序，文件名为：
```
特殊教育多模态干预系统 Setup v1.1.0.exe
```

#### 步骤2：安装应用

1. 双击运行安装程序
2. 选择安装目录（默认：`C:\Program Files\特殊教育多模态干预系统\`）
3. 点击"安装"按钮
4. 等待安装完成（约1-2分钟）
5. 点击"完成"退出安装向导

#### 步骤3：首次启动与激活

1. 双击桌面快捷方式启动应用
2. 应用首次启动会显示"激活页面"
3. 记录页面上的"机器码"（一串64位字符）
4. 联系供应商获取对应激活码
5. 在激活页面输入激活码，点击"激活"按钮
6. 激活成功后，系统自动跳转至主界面

> **提示**：机器码与您的计算机硬件绑定，更换计算机需要重新激活。

### 首次登录配置向导

系统支持自定义以下配置（可选）：

1. **产品名称**：可修改系统显示名称（如改为学校或机构名称）
2. **Logo设置**：可更换为学校或机构Logo
3. **管理员密码**：用于访问高级设置功能

配置入口：点击主界面左上角Logo图标，输入默认管理员密码后进入高级设置。

**紧急预案**：

#### 场景1：激活失效

**症状**：启动后显示激活页面，但输入激活码后报错

**排查步骤**：
1. 检查硬件是否有变更（更换网卡、硬盘等）
2. 记录新的机器码
3. 联系供应商更新激活码

**解决方案**：
- 使用新机器码获取新激活码
- 或还原到原始硬件环境

#### 场景2：应用无法启动

**症状**：点击应用卡片无反应或报错

**排查步骤**：
1. 检查应用路径是否正确（查看apps.json）
2. 检查应用文件是否存在
3. 查看日志文件（logs/error.log）

**解决方案**：
- 修正apps.json中的路径
- 重新部署应用文件
- 联系供应商获取应用更新

#### 场景3：系统崩溃

**症状**：应用启动时闪退或运行中崩溃

**排查步骤**：
1. 查看日志文件中的ERROR记录
2. 检查磁盘空间是否充足
3. 检查是否有冲突的软件

**解决方案**：
- 清理缓存和数据目录
- 卸载后重新安装
- 联系供应商技术支持

## 常见问题与解答

### Q1：如何获取机器码和激活码？

**A**：
1. 启动应用后，系统会自动在激活页面显示机器码（64位字符）
2. 将机器码发送给供应商
3. 供应商根据机器码生成对应的激活码
4. 在激活页面输入激活码，点击"激活"按钮

### Q2：更换电脑后需要重新激活吗？

**A**：需要。机器码与硬件绑定，更换电脑后机器码会发生变化。请联系供应商重新获取激活码。

### Q3：忘记管理员密码怎么办？

**A**：
1. 联系供应商获取重置工具
2. 或卸载应用后重新安装（需重新激活）
3. 重要提示：重置管理员密码会清除所有配置信息

### Q4：可以在多台电脑上使用同一个激活码吗？

**A**：不可以。每个激活码只能在一台电脑上使用。如需在多台电脑安装，请联系供应商获取多个激活码。

### Q5：使用统计数据会自动上传到云端吗？

**A**：不会。所有使用统计数据均存储在本地，不会自动上传，保护隐私。如需云端管理，请点击"系统数据管理"模块手动上传。

### Q6：系统支持哪些Windows版本？

**A**：支持Windows 10及以上版本（32位和64位）。暂不支持Windows 7及以下版本。

### Q7：应用启动时显示"应用路径无效"怎么办？

**A**：
1. 检查应用安装目录下的flash文件夹是否存在
2. 检查apps.json中的应用路径是否正确
3. 重新安装应用或联系供应商

### Q8：如何查看详细的使用日志？

**A**：
1. 日志文件位于：`%APPDATA%\特殊教育多模态干预系统\logs\`
2. 使用记事本或其他文本编辑器打开.log文件
3. 如需技术支持，请将error.log文件发送给供应商

### Q9：系统支持多语言吗？

**A**：目前仅支持简体中文。未来版本可能会增加多语言支持。

### Q10：如何更新系统到最新版本？

**A**：
1. 联系供应商获取最新版本安装包
2. 备份重要数据（激活码、使用统计等）
3. 卸载旧版本并安装新版本
4. 使用之前的激活码激活新版本

### Q11：可以在Mac或Linux系统上使用吗？

**A**：当前版本仅支持Windows系统。如需其他平台支持，请联系供应商定制开发。

### Q12：系统需要联网才能使用吗？

**A**：基本功能无需联网。激活功能为离线验证，不需要网络连接。仅"系统数据管理"模块需要联网访问云端管理平台。

### Q13：应用列表中的应用数量可以自定义吗？

**A**：可以。管理员可以在高级设置中编辑apps.json文件，添加或删除应用。修改后需重启应用生效。

### Q14：系统支持批量部署吗？

**A**：支持。IT管理员可以使用静默安装参数批量部署到多台电脑。具体操作请联系供应商获取批量部署指南。

### Q15：如何导出使用统计数据？

**A**：
1. 进入"高级设置"页面
2. 点击"使用统计"标签
3. 选择时间范围和筛选条件
4. 点击"导出"按钮，选择导出格式（Excel/CSV）
5. 数据将保存到本地

## 故障排查速查表

| 现象 | 可能原因 | 排查命令/方法 | 解决方案 |
|-----|---------|--------------|---------|
| 应用无法启动 | 32位/64位架构不匹配 | 检查系统架构 | 下载匹配的安装包 |
| 激活码无效 | 激活码与机器码不匹配 | 记录机器码联系供应商 | 获取正确激活码 |
| 界面显示异常 | 分辨率过低 | 检查屏幕分辨率 | 调整至1024×768以上 |
| 应用启动失败 | 路径配置错误 | 检查apps.json | 修正路径或重装应用 |
| 高级设置无法访问 | 管理员密码错误 | 重新输入或重置密码 | 使用正确的密码 |
| 统计数据丢失 | 数据库文件损坏 | 检查logs目录 | 恢复备份数据 |
| 内存占用过高 | 应用未正常关闭 | 检查任务管理器 | 关闭异常进程并重启 |
| Logo无法显示 | Logo文件损坏 | 检查logos目录 | 重新上传Logo |
| 系统运行缓慢 | 硬件配置不足 | 检查CPU/内存使用率 | 升级硬件或关闭其他程序 |
| 激活信息丢失 | 数据文件被删除 | 检查activation.json | 重新激活系统 |
